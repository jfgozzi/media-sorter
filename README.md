# 📸 Media Sorter Pro

A Python script designed to automate the organization of multimedia directories. It categorizes photos and videos by parsing EXIF metadata and detects visual duplicates using Perceptual Hashing algorithms, safely handling naming collisions and corrupted files.

**The Backstory:** The idea for this script was born from an old flash drive I had at home, containing years' worth of childhood photos. The drive was a mess: duplicate images, rotated versions, inconsistent file names, and proprietary junk files generated by the cameras used back then (mostly a Sony DSC-H9 leaving `.moff` files everywhere). I built this tool to finally sort my childhood photo albums, and while it was tailored for that use case, it easily scales to handle other image and video formats. 😁

## 🚀 Key Features

* **EXIF Metadata Extraction:** Automatically organizes images into chronological folders (`YYYY-MM`) by reading the `DateTimeOriginal` tag using the `exifread` library.
* **Advanced Duplicate Detection (pHash):** Uses Perceptual Hashing (`imagehash`) to identify identical images even if they have been altered or rotated (evaluates matrices at 0°, 90°, 180°, and 270°).
* **Edge Case Handling:**
    * **Safe Auto-Renaming:** Prevents file overwriting (name collisions) by appending incremental suffixes (e.g., `photo_1.jpg`).
    * **Fault Tolerance:** Proactively catches parsing errors from corrupted dates and routes those files to a safe contingency folder (`No_Date`).
    * **Junk File Filtering:** Automatically identifies and separates residual/system files (e.g., `.thm`, `.moff`, `.DS_Store`).
* **Modular Architecture:** Clear separation of concerns (Configurations, Utilities, and Main Loop), making it easy to maintain and ready for future Unit Testing.
